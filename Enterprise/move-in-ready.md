# <a name="move-in-ready"></a>Mover en listo

 **Resumen:** Obtenga los detalles en el conjunto de la nube de opciones de almacenamiento que están integradas en las ofertas de nube de Microsoft.
  
"Mover de preparado" soluciones de almacenamiento de información:
  
- Soluciones de uso envasada inmediatamente
    
- Está preparado para ir, incluido con servicios existentes
    
- Proporcionar personalización limitada
    
Las secciones siguientes describen los detalles de cada almacenamiento "listo para mover en" opción.
  
## <a name="sharepoint-online-in-office-365"></a>SharePoint Online en Office 365

### <a name="features"></a>Características

- Los archivos son seguras, confiables y accesibles desde varios dispositivos y desde cualquier lugar
    
- Contenido es fácilmente reconocible mediante búsqueda
    
- Flujos de trabajo y políticas de retención hacen fácil de utilizar y mantener el contenido
    
- Puede ser el contenido sin conexión o se accede a través de las API de web o en un explorador
    
### <a name="common-uses"></a>Usos comunes

- Portal de la empresa u organización
    
- Colaboración en equipo
    
- Motor de búsqueda Enterprise Search
    
### <a name="key-storage-scenarios"></a>Escenarios de almacenamiento de claves

- Colaborar
    
- Comunicaciones de la empresa
    
- Gestión de vídeo
    
### <a name="resources"></a>Recursos

Para obtener información adicional, haga clic en [aquí](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498).
  
Para obtener información de costo, haga clic en [aquí](https://products.office.com/business/compare-office-365-for-business-plans).

## <a name="azure-backup"></a>Copia de seguridad de Azure

### <a name="features"></a>Características

- Protege sin fisuras de los datos recibidos desde Windows Server y Windows Server Essentials
    
- Incluye System Center Data Protection Manager
    
- Los datos se almacenan en almacenamiento replicado geo, que mantiene seis copias de sus datos a través de dos centros de datos de Azure
    
- Admite cambios delta, frecuencia definida, compresión, cifrado, retención más prolongada y la limitación de ancho de banda
    
### <a name="common-uses"></a>Usos comunes

- Proteger los datos de una gran variedad de orígenes de Azure (VMs) y local (SharePoint, SQL, Exchange)
    
### <a name="key-storage-scenarios"></a>Escenarios de almacenamiento de claves

- Archivos de copia de seguridad
