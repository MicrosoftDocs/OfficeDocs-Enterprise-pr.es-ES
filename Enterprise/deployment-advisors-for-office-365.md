# <a name="deployment-advisors-for-office-365-or-microsoft-365-services"></a>Asesores de implementación para los servicios de Office 365 o Microsoft 365

Los asesores de implementación para Office 365 o Microsoft 365 proporcionan compatibilidad autoguiada para configurar Office 365 para su organización. Para acceder a ellas:

1. Inicie sesión como administrador de inquilinos en el [centro de administración de Microsoft 365](https://admin.microsoft.com).
2. Seleccione uno de los asesores de implementación o las guías que se enumeran a continuación.

Si no inicia sesión como administrador de inquilinos, verá un mensaje en el que necesita tener permisos de administrador de inquilinos.

Cada asesor es un asistente que le guía por las opciones de las características y opciones que desea implementar. En algunos casos, puede usar el Asistente para configurar algo o activar una opción. Aquí le mostramos un ejemplo.

![Ejemplo de un asesor de implementación](./media/deployment-advisors-for-office-365/m365-deploy-advisor1.png)

El asesor le guiará por las decisiones para personalizar sus necesidades. Además, obtendrá instrucciones, scripts y otros recursos para realizar los cambios de configuración.

Al final, obtendrá vínculos a vídeos y artículos de referencia. Aquí le mostramos un ejemplo.

![Ejemplo de vínculos a vídeos y artículos de referencia](./media/deployment-advisors-for-office-365/m365-deploy-advisor2.png)

Puede usar estos asesores en cualquier momento para obtener más información sobre las características y opciones de Office 365. Esto incluye durante la planeación o después de haber configurado los servicios.
  
## <a name="microsoft-365-windows-10-with-office-365"></a>Microsoft 365: Windows 10 con Office 365

Microsoft 365 es una solución completa e inteligente que incluye Office 365, Windows 10 y Enterprise Mobility + Security (EMS). Microsoft 365 permite a todo el mundo ser creativo y trabajar juntos, de forma segura. 

Use este asesor para configurar dispositivos con Windows 10 o actualizar los equipos de los usuarios a Windows 10, con una implementación opcional de las aplicaciones de Office, Windows Analytics y la protección contra amenazas avanzada de Windows Defender (ATP) (solo para planes de Microsoft 365 Enterprise E5).

## <a name="communication-and-online-conferencing"></a>Comunicación y conferencias en línea

### <a name="office-365-groups"></a>Grupos de Office 365

Office 365 los grupos son un área de trabajo compartida para el correo electrónico, las conversaciones, los archivos y los eventos en los que los miembros del grupo pueden realizar las cosas colectivamente. Grupos en Office 365 permite elegir un conjunto de personas con las que desea colaborar y configurar fácilmente una colección de recursos para que las personas compartan. No tiene que preocuparse de asignar manualmente permisos a todos esos recursos, ya que al agregar miembros al grupo, se les asignan automáticamente los permisos que necesitan para las herramientas que proporciona el grupo.
