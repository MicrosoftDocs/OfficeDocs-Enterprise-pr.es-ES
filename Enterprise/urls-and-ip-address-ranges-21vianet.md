- <span data-ttu-id="a457e-p106">**ID**: el número de identificación de la fila, también conocido como un conjunto de puntos de conexión. This ID is the same as is returned by the web service for the endpoint set.</span></span>

- <span data-ttu-id="a457e-p107">**Categoría**: muestra si el conjunto de puntos de conexión se clasifica como "Optimizar", "Permitir" o "Predeterminado". - <span data-ttu-id="a457e-p108">**EMERGENCIA**: aparece como **Sí** si el conjunto de puntos de conexión se admite en Azure ExpressRoute con prefijos de ruta de Office 365. However, it should not be assumed that no routes are advertised for an endpoint set where ER is **No**.</span></span>

- <span data-ttu-id="a457e-p109">**Direcciones**: enumera los FQDN o nombres de dominio con caracteres comodín y los intervalos de direcciones IP para el conjunto de puntos de conexión.
